# Решение задачи

Программа запрашивает у пользователя положительное целое число, 
проверяет корректность ввода и вычисляет факториал.

## Возможности

- ввод числа через консоль
- проверка корректности ввода
- обработка ошибок
- вычисление факториала с использованием библиотеки `math`

## Запуск

Перейдите в папку `case-2` и выполните команду:

```bash
python main.py
