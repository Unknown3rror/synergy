# Кейс-задача № 2 — Вычисление факториала

## Описание

Программа запрашивает у пользователя положительное целое число,  
выполняет проверку корректности ввода и вычисляет факториал введённого числа.

## Требования, выполненные в программе

- Запрос положительного целого числа у пользователя
- Обработка ошибок:
  - Ввод нечисловых данных
  - Ввод отрицательных чисел
  - Ввод пустой строки
- Повторный запрос ввода при ошибке
- Оптимизированное вычисление факториала с помощью `math.factorial()`
- Понятные сообщения об ошибках

## Запуск программы

1. Убедитесь, что у вас установлен Python 3.
2. Перейдите в папку с программой.
3. Выполните команду:

```bash
python main.py
