# Учебная практика (Python)

В данном репозитории будут храниться решения задач,
выполненных в рамках учебной практики.

## Структура проекта

Проект разделен на отдельные директории.  
Каждая директория содержит самостоятельное решение с исходным кодом и кратким описанием.

Внутри каждой папки находятся:
- исходный код (`main.py`)
- описание и инструкция по запуску (`README.md`)

## Запуск

Перейдите в нужную директорию и выполните:

```bash
python main.py
