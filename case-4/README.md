# Кейс-задача № 4 — Игра "Угадай число"

## Описание
Консольная игра, в которой компьютер загадывает случайное число от 1 до 100,  
а пользователь должен угадать его за ограниченное количество попыток.

## Выполненные требования
- Случайная генерация числа от 1 до 100
- Запрос предположения у пользователя
- Подсказки "Слишком маленькое" / "Слишком большое"
- Ограничение на 10 попыток
- Обработка некорректного ввода (нечисловые значения, числа вне диапазона)
- Возможность сыграть несколько раз
- Понятный и дружелюбный интерфейс

## Запуск программы
1. Убедитесь, что у вас установлен Python 3.
2. Перейдите в папку с программой (`case-4`).
3. Выполните команду:

```bash
cd case-4
python main.py
